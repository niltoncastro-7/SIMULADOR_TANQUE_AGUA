Diagrama sinóptico del proceso

El tanque se descarga automáticamente por consumo simulado. Al llegar al nivel mínimo, se activa la bomba de entrada. Cuando llega al nivel máximo, la bomba se apaga y el tanque vuelve a descargarse.

Secuencia automática

Paso 1El tanque baja por descarga simulada.
Paso 2Cuando llega al mínimo, inicia llenado automático.

ProtecciónParada de emergencia bloquea el sistema hasta reinicio.

Control de operación

En automático, el tanque se vacía solo y se llena al llegar al mínimo. En manual, el operador puede forzar llenado o descarga.

Software utilizado

HTML, CSS y JavaScript en un solo archivo index.html. Puede ejecutarse localmente o en GitHub Pages.

Proceso simulado

Sistema HMI/SCADA para tanque de agua con descarga automática y llenado automático por bajo nivel.

Variables principales

Nivel, temperatura, caudal, presión, energía, volumen acumulado, bomba, válvulas y ciclo de operación.

Alarmas

Nivel alto, nivel bajo, temperatura alta, sobrepresión, sistema detenido, falla de caudal y parada de emergencia.

Controles

Inicio, parada, modo automático/manual, llenado manual, descarga manual, emergencia, reconocimiento de alarmas, setpoints, velocidad y fallas de prueba.

Conclusión

El prototipo muestra una interfaz industrial completa con monitoreo, control, alarmas, tendencias, historial, mantenimiento y reporte.
